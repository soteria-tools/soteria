Module Bv_values.Typed

module type S = sig ... end
module Make_transparent (V : Svalue.Value_ext) () : sig ... end

Like Make, but additionally exposes the ghost-typed +'a t/+'a ty as transparently equal to the underlying untyped svalue. This lets a downstream module write extension helpers without juggling type_/untyped/cast, while its own .mli re-seals t/ty as abstract for the rest of the world.

module Make (V : Svalue.Value_ext) () : S with module Ext = V

The main entry point of Bv_values. Builds the ghost-typed svalue layer over Svalue.Make: the same hash-consed values, but exposed through a phantom-typed +'a t whose 'a tracks each value's kind (the sint, sbool, sptr, ... tags) so that ill-kinded combinations are rejected at compile time. The result also bundles the matching Svalue (untyped layer), Eval (normalisation) and Expr (substitution) modules.
