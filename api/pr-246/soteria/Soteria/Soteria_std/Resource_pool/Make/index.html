val create_pool : unit -> pool

Create a new empty pool.

val acquire : pool -> t * auth

Acquire a resource from the pool. If an unused resource is available, it is returned. Otherwise, a new resource is created. Returns the pooled resource and an auth token that must be used for subsequent operations.

val release : pool -> auth -> t -> unit

Release a resource back to the pool. The auth token must match the one returned by acquire. Raises Pool_invalid_auth if the auth token is invalid.

val handle : t -> int

Get the handle of the pooled resource. It is a unique identifier for the resource within the pool.

val apply : auth:auth -> t -> (resource -> 'a) -> 'a

Access the underlying resource. The auth token must match the current valid token for this resource. Raises Pool_invalid_auth if the resource has been released.
