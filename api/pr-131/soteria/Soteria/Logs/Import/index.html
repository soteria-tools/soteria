Module Logs.Import

Helper module to be opened when using logging.

  open Soteria.Logs.Import
  ...

  L.info (fun m -> m "Hello world")
module L : module type of L

L is the module that contains all the fonctions to actually log stuff. One can access it using Soteria.Logs.L or by open Soteria.Logs.Import and then using L directly.
