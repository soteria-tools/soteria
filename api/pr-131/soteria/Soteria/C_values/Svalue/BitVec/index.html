Bit vectors

Bit vector operations; all these expect values of type TInt, and apply bit-vector conversions under the hood as needed. This allow bit operations to be seamlessly used on TInt, without worrying about the underlying representation.

val bool_or : t -> t -> t

module Raw : sig ... end

Raw operations for values of type TBitVector; be careful when using these, so as to provide properly typed values. Prefer using BitVec directly when possible, as it may also provide more reductions.

val contains_bvs : t -> (bool * int) option

Some (signed, size) if the given value contains somewhere some bitvector operations; None otherwise.
