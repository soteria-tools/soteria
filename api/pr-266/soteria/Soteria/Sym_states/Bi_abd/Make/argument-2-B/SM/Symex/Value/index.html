Module Symex.Value

type 'a t = 'a Symex.Value.t

The type of symbolic values, parameterized by their type. For example, sbool t represents a symbolic boolean.

type 'a ty = 'a Symex.Value.ty

Type of values.

type sbool = Symex.Value.sbool

The type for booleans.

val not : sbool t -> sbool t

not b returns the logical negation of boolean b.

val ppa : Stdlib.Format.formatter -> 'a t -> unit
val iter_vars : 'a t -> 'b ty Symex.Var.iter_vars

iter_vars v f iterates over all variables in value v, calling f (var, ty) for each variable with its type.

val subst : (Symex.Var.t -> Symex.Var.t) -> 'a t -> 'a t

subst f v substitutes variables in v using function f. Each variable x is replaced with f x.

val mk_var : Symex.Var.t -> 'a ty -> 'a t

mk_var var ty creates a symbolic variable with name var and type ty.

val as_bool : 'a t -> bool option

as_bool v returns Some b if v is a concrete boolean b, or None if v is symbolic.

val bool : bool -> sbool t

bool b creates a concrete boolean value from b.
