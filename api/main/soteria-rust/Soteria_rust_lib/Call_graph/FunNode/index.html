Module Call_graph.FunNode

Node type: a fully-qualified function name rendered as a string.

type t = {
  1. short_name : string;
  2. long_name : string;
}
val equal : t -> t -> bool
val hash : t -> int
val pp : Stdlib.Format.formatter -> t -> unit
val long_name : t -> string
val short_name : t -> string
val short_name_of_name : Charon.Types.path_elem list -> Charon.Types.path_elem list

The "short name" is the last 2 identifiers (excluding type parameters e.g. `<T>`), if there are two, otherwise just the last identifier (should always be the case), fallbacks to the full name.
