General definitions

type stat_entry =
  1. | Int of int
    (*

    An integer statistic; joined by addition.

    *)
  2. | Float of float
    (*

    A float statistic; joined by addition.

    *)
  3. | StrSeq of string Stdlib.Dynarray.t
    (*

    A sequence of strings; joined by concatenation.

    *)
  4. | Map of stat_entry Soteria.Soteria_std.Hashtbl.Hstring.t
    (*

    A map from strings to stat entries; joined by merging entries recursively.

    *)
  5. | Yojson of Yojson.Safe.t
    (*

    A JSON value; joined by creating a JSON array of the two values if both are not arrays, or concatenating the two arrays if they are. Use this as a last resort when no other type fits.

    *)

Accessors

val as_int : stat_entry -> int

as_int entry returns the integer value of the statistic entry entry. Raises Incompatible_entries if entry is not an integer.

val as_float : stat_entry -> float

as_float entry returns the float value of the statistic entry entry. Raises Incompatible_entries if entry is not a float.

val as_strseq : stat_entry -> string Stdlib.Dynarray.t

as_strseq entry returns the string sequence value of the statistic entry entry. Raises Incompatible_entries if entry is not a string sequence.

val as_map : stat_entry -> stat_entry Soteria.Soteria_std.Hashtbl.Hstring.t

as_map entry returns the map value of the statistic entry entry. Raises Incompatible_entries if entry is not a map.

val as_yojson : stat_entry -> Yojson.Safe.t

as_yojson entry returns the JSON value of the statistic entry entry. Raises Incompatible_entries if entry is not a JSON value.

val get_entry : t -> string -> stat_entry option

get_entry stats name returns the statistic entry under the given name name in the statistics collection stats, or None if no such entry exists.

val get_int : ?default:int -> t -> string -> int

get_int ?default stats name returns the integer statistic under the given name name in the statistics collection stats. If no such entry exists, returns default if provided (defaults to 0). Raises Incompatible_entries if the entry under name is not an integer.

val get_float : ?default:float -> t -> string -> float

get_float ?default stats name returns the float statistic under the given name name in the statistics collection stats. If no such entry exists, returns default if provided (defaults to 0.0). Raises Incompatible_entries if the entry under name is not a float.

val get_strseq : t -> string -> string Stdlib.Dynarray.t

get_strseq stats name returns the string sequence statistic under the given name name in the statistics collection stats. If no such entry exists, returns an empty sequence. Raises Incompatible_entries if the entry under name is not a string sequence.

val get_map : t -> string -> stat_entry Soteria.Soteria_std.Hashtbl.Hstring.t

get_map stats name returns the map statistic under the given name name in the statistics collection stats. If no such entry exists, returns an empty map. Raises Incompatible_entries if the entry under name is not a map.

val register_printer : string -> ?name:string -> (t -> Stdlib.Format.formatter -> stat_entry -> unit) -> unit

register key ?name printer Registers a new printer for statistics entries of name name, possibly overriding previously set printers. If an entry with the given name is found during pretty-printing and no printer has been registered for that name, default_printer is used instead. printer also receives the full stats object, to enable more context-aware printing. A string name can be provided to be used as a header when printing the statistic; if none is provided, key is used instead.

val disable_printer : string -> unit

disable_printer key disables any printing for the statistic for key, meaning it will not be printed at all. Use register_printer to revert this.
