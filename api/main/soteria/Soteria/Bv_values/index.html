Module Soteria.Bv_values

Extensible bit-vector based values, with floating point support. Suitable for low-level languages.

  • Overview
  • Using the functors

Overview

The core is Svalue: an untyped, hash-consed expression grammar (booleans, bit-vectors, floats, pointers, sequences, quantifiers) equipped with simplifying smart constructors. On top of it Typed adds a phantom "ghost" type tracking the kind of each value (integer, boolean, pointer, ...) for extra static safety. Encoding lowers values to SMT, Bv_solver drives Z3, and Analyses are the abstract domain analyses used along the way.

Using the functors

Everything is parameterised by Value_ext, a domain-specific extension that adds extra value/type leaves to the grammar. Tools that need no extra leaves pass Svalue.Dummy_ext.

A tool instantiates the typed layer once and reuses it everywhere:

  module Typed = Bv_values.Typed.Make (Bv_values.Svalue.Dummy_ext) ()

From Typed one then gets Typed.Svalue (the untyped layer and its constructors), Typed.Eval (normalisation) and Typed.Expr (substitution). To discharge path conditions, feed Typed to a solver from Bv_solver.
