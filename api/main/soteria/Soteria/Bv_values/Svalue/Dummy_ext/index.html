Module Svalue.Dummy_ext

The empty extension: t and ty are uninhabited, so an svalue built with Dummy_ext never contains an Extension node nor a TExtension type, and every operation above is vacuous. This is the right choice for tools (such as soteria-c and soteria-rust) whose values need nothing beyond the built-in bit-vector / float / pointer grammar.
