Module Svalue.Make

Builds the untyped svalue layer for extension V: the value/type definitions specialised to V, the hash-cons table, and the simplifying smart constructors (Bool, BitVec, Float, Ptr, SSeq, ...). This is the foundation the rest of Bv_values is built on; most callers reach it as Typed.Svalue rather than applying it directly (see Typed.Make).

Make is generative (note the final ()): each application mints a fresh, abstract ghost tagging every value it hash-conses, and owns its own table. Two applications — even to the same V — therefore produce mutually incompatible ts, so values from one can never be mixed with another's. Apply it once and share the resulting module.
