Whether the given type is monomorphic, i.e. contains no type variables. This is a conservative estimate: struct Foo<T> {} is considered polymorphic despite the generics being unused.

val tyref_is_substituted : Charon.Types.type_decl_ref -> bool

Whether the given type decl ref is properly substituted, i.e. it contains no bound type variables. This can still return true for polymorphic type decl refs, e.g. Option<@Free0>, but we guarantee all type variables are free.
